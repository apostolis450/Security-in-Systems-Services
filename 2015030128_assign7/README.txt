Χρησιμοποιούμε την εντολή iptables ώστε να κάνουμε τις αλλαγές που
ζητούνται.Κάνοντας dig +short σε κάθε domain name που δίνεται στο αντίστοιχο
αρχείο, παίρνουμε την ip του domain, αλλά μπορεί να υπάρχει και ένας ενδιάμεσος
domain,οπότε γίνεται ξανα resolve με την dig σε αυτή την περίπτωση.
Οι ip αυτές γράφονται σε ένα αρχείο το οποίο σβήνεται μετα την δημιουργία των κανόνων , και έπειτα 
δημιουργούμε τους κανόνες με αυτές τις ip.Συγκεκριμένα,
κάνουμε reject κάθε request (output chain) που κάνει ένας ιστότοπος στις συγκεκριμένες
ip για να προβάλει την διαφήμιση.
Δοκιμάστηκαν οι κανόνες στα speedtest.net, dailymail.co.uk, με κλειστό adblock extention
και λειτουργούν όπως πρέπει.Σε κάποια site  υπήρχε μια έξτρα καθυστέρηση στην
φόρτωση της σελίδας. Αυτό μπορεί να συμβαίνει γιατί κάποια site μπορεί να κάνουν
self-host τις διαφημισεις, οπότε αν μπλοκάρουμε τον domain δε θα μπορεί  να φορτωσει το site.
